Historique et popularité de Python

Python est un langage de programmation de haut niveau, interprété, multiplateforme et open source. Il a été créé par Guido van Rossum en 1991 et est depuis devenu l'un des langages de programmation les plus populaires au monde. Python est connu pour sa syntaxe claire et concise, sa grande bibliothèque standard et sa communauté active.

Utilisations et applications de Python

Python est utilisé dans de nombreux domaines, notamment :

  • Développement web : Python est utilisé pour créer des sites web et des applications web dynamiques avec des frameworks comme Django, Flask et Pyramid.
  • Intelligence artificielle et apprentissage automatique : Python est devenu le langage de choix pour l'IA et le ML grâce à des bibliothèques comme TensorFlow, Keras et scikit-learn.
  • Analyse de données et science des données : Python est largement utilisé pour l'analyse de données et la visualisation avec des bibliothèques comme NumPy, Pandas et Matplotlib.
  • Développement de logiciels : Python est utilisé pour développer des logiciels de bureau et des applications mobiles avec des outils comme PyQt, Kivy et BeeWare. Programmation système et administration : Python est utilisé pour automatiser des tâches, gérer des fichiers et interagir avec le système d'exploitation.
  • Jeux vidéo : Python est utilisé pour développer des jeux vidéo avec des bibliothèques comme Pygame et Panda3D.

Environnement de développement et installation de Python

Pour commencer à programmer en Python, vous devez installer un environnement de développement et Python lui-même. Voici quelques étapes de base :
  1. Téléchargez et installez Python depuis le site officiel : https://www.python.org/downloads/
  2. Installez un éditeur de texte ou un environnement de développement intégré (IDE) pour écrire et exécuter du code Python. Quelques options populaires incluent Visual Studio Code, PyCharm, et Atom.
  3. Configurez votre environnement de développement pour qu'il reconnaisse Python. Cela peut varier en fonction de l'éditeur ou de l'IDE que vous utilisez.

Une fois que vous avez installé Python et configuré votre environnement de développement, vous êtes prêt à commencer à apprendre et à écrire du code Python.

Commentaires et indentation

Les commentaires en Python sont utilisés pour expliquer le code et sont ignorés lors de l'exécution. Un commentaire commence par le caractère '#' et se poursuit jusqu'à la fin de la ligne.

# Ceci est un commentaire

L'indentation en Python est utilisée pour définir les blocs de code. Contrairement à d'autres langages qui utilisent des accolades ou des mots-clés, Python utilise des espaces pour indiquer le début et la fin des blocs de code. Il est important de maintenir une indentation cohérente pour éviter les erreurs.

if condition:
        # Indentation pour le code de la condition
        print("La condition est vraie")

Variables et types de données

En Python, vous pouvez stocker des valeurs dans des variables. Les types de données courants en Python sont :

  • int : entiers (par exemple, 1, 2, 3)
  • float : nombres à virgule flottante (par exemple, 1.5, 2.2, 3.14)
  • bool : booléens (True, False)
  • str : chaînes de caractères (par exemple, "bonjour", 'hello')
  • list : listes (par exemple, [1, 2, 3])
  • tuple : tuples (par exemple, (1, 2, 3))
  • dict : dictionnaires (par exemple, {"clé": "valeur"})
  • set : ensembles (par exemple, {1, 2, 3})

Voici quelques exemples de déclaration de variables et d'affectation de valeurs :


    x = 1  # x est un entier (int)
    y = 3.14  # y est un nombre à virgule flottante (float)
    z = "bonjour"  # z est une chaîne de caractères (str)

  • Opérateurs de comparaison : == (égalité), != (inégalité), > (supérieur), < (inférieur), >= (supérieur ou égal), <= (inférieur ou égal)

Conditions

Les conditions en Python sont utilisées pour exécuter différents blocs de code en fonction de la valeur de certaines expressions. Les conditions sont définies à l'aide des mots-clés if, elif (pour "else if") et else.


  age = 18
  if age < 18:
      print("Vous êtes mineur.")
  elif age < 65:
      print("Vous êtes majeur.")
  else:
      print("Vous êtes senior.")

Boucles

Les boucles en Python sont utilisées pour exécuter un bloc de code plusieurs fois. Il existe deux types de boucles : for et while.

La boucle for itère sur une séquence (par exemple, une liste, une chaîne de caractères, un tuple) et exécute le bloc de code pour chaque élément de la séquence.


  fruits = ["pomme", "banane", "orange"]
  for fruit in fruits:
      print("J'aime manger des", fruit)

La boucle while exécute un bloc de code tant qu'une condition est vraie.


  i = 0
  while i < 5:
      print(i)
      i += 1

Gestion des exceptions

Les exceptions en Python sont utilisées pour gérer les erreurs et les situations imprévues. La structure try, except et finally est utilisée pour gérer les exceptions. 


  try:
      # Code susceptible de générer une exception
      result = 10 / 0
  except ZeroDivisionError:
      # Code à exécuter si une exception ZeroDivisionError est levée
      print("Erreur : division par zéro.")
  else:
      # Code à exécuter si aucune exception n'est levée
      print("Le résultat est", result)
  finally:
      # Code à exécuter indépendamment du fait qu'une exception soit levée ou non
      print("Fin de la gestion des exceptions.")

Définition de fonctions

Une fonction en Python est un bloc de code réutilisable qui effectue une tâche spécifique. Les fonctions sont définies à l'aide du mot-clé def.

def saluer(nom):

Arguments et paramètres

Les fonctions peuvent accepter des arguments (ou paramètres) qui sont utilisés dans le corps de la fonction. Les arguments peuvent être de n'importe quel type de données et peuvent être nombreux, séparés par des virgules. 

def additionner(a, b):
    return a + b

Portée des variables

En Python, les variables ont une portée locale ou globale. Les variables définies à l'intérieur d'une fonction sont locales et ne sont accessibles qu'à l'intérieur de cette fonction. Les variables définies en dehors d'une fonction sont globales et peuvent être utilisées partout dans le programme.

x = 10  # Variable globale

def fonction():
    y = 20  # Variable locale
    print(x, y)  # Affiche 10, 20

fonction()
print(x, y)  # Affiche 10, erreur car y n'est pas défini en dehors de la fonction

Slicing et indexation

Le slicing et l'indexation permettent d'accéder à des éléments spécifiques dans une liste ou un tuple. L'indexation commence à 0, donc le premier élément d'une liste ou d'un tuple est à l'index 0.

liste = [1, 2, 3, 4, 5]
print(liste[0])  # Affiche 1 (premier élément de la liste)
print(liste[2:4])  # Affiche [3, 4] (troisième et quatrième éléments de la liste)

Fonctions built-in pour les listes et tuples

liste = [1, 2, 3, 4, 5]

liste.remove(3)  # Supprime le premier élément 3 de la liste
print(liste.index(4))  # Affiche 3 (index du premier élément 4 dans la liste)

Création et manipulation de dictionnaires et ensembles

Les dictionnaires et les ensembles en Python sont des structures de données non ordonnées et modifiables qui contiennent des éléments de différents types. Les dictionnaires sont constitués de paires clé-valeur, tandis que les ensembles ne contiennent que des valeurs uniques.

dictionnaire = {"clé1": "valeur1", "clé2": "valeur2"}
ensemble = {1, 2, 3, 4, 5}

Accès aux éléments et itération

Les dictionnaires et les ensembles permettent d'accéder à des éléments spécifiques et d'itérer sur leurs éléments.

dictionnaire = {"clé1": "valeur1", "clé2": "valeur2"}
print(dictionnaire["clé1"])  # Affiche "valeur1"

ensemble = {1, 2, 3, 4, 5}
for element in ensemble:
    print(element)  # Affiche 1, 2, 3, 4, 5

Fonctions built-in pour les dictionnaires et ensembles

Python fournit de nombreuses fonctions built-in pour manipuler les dictionnaires et les ensembles. Quelques exemples incluent keys(), values(), items(), add(), remove(), update(), pop(), etc. 

dictionnaire = {"clé1": "valeur1", "clé2": "valeur2"}
print(dictionnaire.keys())  # Affiche dict_keys(['clé1', 'clé2'])
print(dictionnaire.values())  # Affiche dict_values(['valeur1', 'valeur2'])
print(dictionnaire.items())  # Affiche dict_items([('clé1', 'valeur1'), ('clé2', 'valeur2')])

ensemble = {1, 2, 3, 4, 5}
ensemble.add(6)  # Ajoute 6 à l'ensemble
ensemble.remove(3)  # Supprime 3 de l'ensemble

dictionnaire.update({"clé3": "valeur3"})  # Ajoute la paire clé-valeur "clé3": "valeur3" au dictionnaire
valeur = dictionnaire.pop("clé1")  # Supprime la paire clé-valeur "clé1": "valeur1" du dictionnaire et renvoie "valeur1"

Création de classes et d'objets

La programmation orientée objet (POO) en Python permet de créer des classes et des objets. Les classes sont des modèles à partir desquels les objets sont créés. Les objets sont des instances de classes.

class Animal:
    def __init__(self, nom, espece):
        self.nom = nom
        self.espece = espece

    def se_presenter(self):
        print("Je suis", self.nom, "et je suis un", self.espece)

mon_animal = Animal("Fido", "chien")
mon_animal.se_presenter()  # Affiche "Je suis Fido et je suis un chien"

Attributs et méthodes

Les attributs sont des variables associées à une classe ou un objet, et les méthodes sont des fonctions associées à une classe ou un objet.

class Personne:
    def __init__(self, nom, age):
        self.nom = nom
        self.age = age

    def dire_bonjour(self):
        print("Bonjour, je m'appelle", self.nom)

moi = Personne("Alice", 30)
moi.dire_bonjour()  # Affiche "Bonjour, je m'appelle Alice"

Héritage et polymorphisme

L'héritage permet de créer de nouvelles classes à partir de classes existantes. Le polymorphisme permet d'utiliser des méthodes de différentes classes de la même manière.

class Animal:
    def __init__(self, nom):
        self.nom = nom

    def dire_bonjour(self):
        print("Bonjour, je suis", self.nom)

class Chien(Animal):
    def dire_bonjour(self):
        print("Wouf ! Je suis", self.nom)

mon_animal = Chien("Fido")
mon_animal.dire_bonjour()  # Affiche "Wouf ! Je suis Fido"

Méthodes spéciales

Les méthodes spéciales en Python commencent et se terminent par des doubles underscores (__). Quelques exemples incluent __init__, __str__, __repr__, etc.

class Personne:
    def __init__(self, nom, age):
        self.nom = nom
        self.age = age

    def __str__(self):
        return "Personne(nom={}, age={})".format(self.nom, self.age)

moi = Personne("Alice", 30)
print(moi)  # Affiche "Personne(nom=Alice, age=30)"

Les fonctions lambda en Python sont des fonctions anonymes à une seule expression. Elles sont utiles pour créer des fonctions simples et courtes.

Compréhensions de listes, dictionnaires et ensembles

Les compréhensions en Python sont des expressions qui permettent de créer des listes, des dictionnaires et des ensembles en une seule ligne de code.

# Compréhension de liste
liste = [x * 2 for x in range(5)]
print(liste)  # Affiche [0, 2, 4, 6, 8]

# Compréhension de dictionnaire
dictionnaire = {x: x * 2 for x in range(5)}

# Compréhension d'ensemble
ensemble = {x * 2 for x in range(5)}

Fonctions map, filter et reduce

Les fonctions map(), filter() et reduce() en Python permettent de manipuler des listes et des itérables de manière fonctionnelle.

# map()
liste = [1, 2, 3, 4, 5]
liste_doublee = list(map(lambda x: x * 2, liste))
print(liste_doublee)  # Affiche [2, 4, 6, 8, 10]

# filter()
liste = [1, 2, 3, 4, 5]
liste_paires = list(filter(lambda x: x % 2 == 0, liste))
print(liste_paires)  # Affiche [2, 4]

# reduce()
from functools import reduce
liste = [1, 2, 3, 4, 5]
somme = reduce(lambda x, y: x + y, liste)
print(somme)  # Affiche 15

Coroutines et générateurs

Les coroutines et les générateurs en Python sont des fonctions spéciales qui peuvent être suspendues et reprises à des points spécifiques. Les coroutines sont des fonctions asynchrones qui peuvent être utilisées pour écrire du code asynchrone.


import asyncio

async def factorielle(n):
    resultat = 1
    for i in range(1, n + 1):
        await asyncio.sleep(0)
        resultat *= i
    return resultat

async def main():
    print("Calcul de la factorielle de 5 :", await factorielle(5))

asyncio.run(main())

asyncio et programmation concurrente

Le module asyncio en Python fournit des outils pour écrire du code asynchrone et concurrent. Il est particulièrement utile pour les applications réseau et les applications qui doivent gérer plusieurs tâches en parallèle.


import asyncio

async def tache1():
    print("Début de la tâche 1")
    await asyncio.sleep(1)
    print("Fin de la tâche 1")

async def tache2():
    print("Début de la tâche 2")
    await asyncio.sleep(2)
    print("Fin de la tâche 2")

async def main():
    await asyncio.gather(tache1(), tache2())

asyncio.run(main())
